# V2.0.0 原型存档

> 存档日期：2026-06-09  
> 用途：冻结当前前端原型，作为新版本功能设计的基线参考。

## 版本定位

- **版本号**：`2.0.0`
- **性质**：具身智能数据平台前端交互原型（Vite + React 19 + TypeScript）
- **数据**：页面内 mock / 本地状态，不依赖后端

## 已覆盖模块

| 路由 | 页面 | 说明 |
|------|------|------|
| `/` | 首页 / 运营仪表盘 | 核心指标、趋势图、传输与设备、告警 |
| `/requirements` | 数据需求 | 需求列表与筛选 |
| `/tasks` | 任务管理 | 任务看板与状态流转 |
| `/resources/*` | 资源管理 | 设备 / 人员 / 场景 |
| `/collection` | 数据采集 | 采集任务与进度 |
| `/collection/app-demo` | 采集端 Demo | 移动端采集界面示意 |
| `/processing` | 数据处理 | 处理流水线 |
| `/annotation` | 数据标注 | 标注任务 |
| `/quality` | 质量准出 | 质检与准出 |
| `/monitor` | 监控大屏 | 全屏监控视图 |

## 如何回溯本版本

### Git

```bash
git checkout archive/v2.0.0-prototype
# 或
git checkout v2.0.0
```

### 本地运行

```bash
npm ci
npm run dev
```

详见 `prototype-handoff/本地运行说明.md`。

从 `main` 分支继续迭代；本存档分支与 tag 只读保留，不随新功能提交。
